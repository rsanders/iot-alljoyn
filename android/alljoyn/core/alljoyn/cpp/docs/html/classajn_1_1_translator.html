virtual const char* ajn::Translator::Translate ( const char *  sourceLanguage,
const char *  targetLanguage,
const char *  source,
qcc::String &  buffer 
) [inline, virtual]

Translate source from sourceLanguage into targetLanguage.

If this Translator does not have a translation for the given parameters it should return NULL. This version of the function is designed for imeplementations that return dynamically allocated strings. The string should be copied into buffer and buffer.c_str() must be returned. When alljoyn finishes using the string buffer will be free'ed

Parameters:
sourceLanguageThe language tag of the text in source
targetLanguageThe language tag to translate into
sourceThe source text to translate
bufferA buffer to hold the dynamically allocated string returned by this function
Returns:
The translated text or NULL
